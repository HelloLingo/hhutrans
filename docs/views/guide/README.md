## Tensorflow使用
### 类别
- 语言感情偏向分类  
能辨别一句话是积极或消极的
- 图像分类  
能分辨一张图像的主体是猫或狗，识别0-9数字图片
- 目标检测  
检测到车辆在图片中的位置并框选
- 目标语义分割  
检测到图片中哪些像素区域代表人，哪些代表车
- 实例分割  
在语义分割的基础上能把每个个体区分开来
- 回归分析  
对数据间的关系做回归预测
- 关键点检测  
检测人体的骨骼关键点
